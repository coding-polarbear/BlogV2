# 북극곰의 개발일기
> Second Blog project with Node.JS &amp; MongoDB

# 사용 기술
* Node.JS
* MongoDB
* mongoose
* pug(template Engine)

# 사용 방법
/config/config.js 생성후 jwtTokenAlgorithm과 salt 설정 그 이후
```bash
npm install
```

```bash
npm start
```

# Blog 역사
[BlogV1](http://github.com/purplebeen/BlogV1) : 첫번째 기술블로그,Spring Boot + Bootstrap
